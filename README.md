# StrayOS

## RU / Русский

StrayOS — мой собственный экспериментальный Linux-дистрибутив, созданный на основе Debian-подобной системы.

От Debian здесь в основном используется только `apt`, совместимость с `.deb` пакетами и базовая пакетная система. Всё остальное постепенно переделывается под свой стиль, свои настройки и свои инструменты.

Проект создан для изучения Linux, кастомизации системы, разработки собственных утилит и экспериментов с устройством операционной системы.

### Важно про GNOME

GNOME в StrayOS нужно запускать только безопасной командой:

strayos-gnome

Не рекомендуется запускать GNOME обычными способами, потому что система может работать нестабильно или запуск окружения может пройти неправильно.

EN / English

StrayOS is my own experimental Linux distribution based on a Debian-like system.

The main parts inherited from Debian are mostly apt, .deb package compatibility, and the basic package management system. Everything else is gradually being redesigned with my own style, settings, and tools.

This project was created for learning Linux, customizing the system, developing custom utilities, and experimenting with how operating systems work.

Important GNOME note

GNOME in StrayOS should only be started using the safe command:

strayos-gnome

It is not recommended to start GNOME using regular methods, because the system may become unstable or the desktop environment may not start correctly.

Features
apt support
.deb package compatibility
Custom system settings
Own visual style
Ability to add custom utilities
Suitable for Linux experiments, shell scripts, and system development
Developed as an educational and experimental OS project
Plans
Add a custom installer
Create a branded visual design
Add custom system commands
Build a custom control center
Add a custom package manager on top of apt
Optimize the system for low-end PCs and laptops
Create an installable ISO image
Status

The project is currently in development. Some features may be unstable or change in future versions.

Warning

StrayOS is an experimental distribution. It is not recommended to use it as a main operating system without backing up your data.
